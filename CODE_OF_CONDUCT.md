# 🕊️ 同舟空间社区文明公约 (Citizen Code of Conduct)

为了维护一个健康、向上的学术交流环境，所有参与本仓库（包括 Issue 讨论、代码提交）的用户请遵守以下准则：

## 1. 核心原则
- **学术尊重**：承认并尊重原作者的知识产权。
- **友善交流**：在讨论资料或技术问题时，保持礼貌与客观。
- **互助精神**：鼓励“我为人人，人人为我”的开源分享精神。

## 2. 严禁行为
- **商业用途**：严禁将本站资源打包出售或用于任何盈利行为。
- **违规内容**：禁止发布违反国家法律法规、学校校规或与学术无关的有害信息。
- **人身攻击**：禁止在 Issue 中进行言语谩骂或骚扰他人。
- **隐私泄露**：禁止在未获授权的情况下发布他人或机构的敏感隐私信息。

## 3. 免责与纠纷处理
- **资料质量**：本站不对资源的完整性或准确性做绝对担保，请使用者自行甄别。
- **侵权处理**：若发现资料存在版权争议，请及时通过 GitHub Issue 提交反馈，并提供相关权属证明，管理员将在确认后第一时间下线处理。

## 4. 管理权限
管理员有权根据本准则对违规行为进行处理，包括但不限于：
- 修改或删除违规 Issue。
- 拒绝或关闭违规的 Pull Request。
- 在必要时暂时或永久封禁违规账户的互动权限。

---
**联系方式**：如有任何疑问或投诉，请在 GitHub 提交 Issue。
